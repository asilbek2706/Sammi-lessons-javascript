# Module 2 — Ilg'or JavaScript

Bu modul JavaScript tilining ilg'or mavzularini amaliy misollar orqali o'rgatadi.

---

## 📁 Darslar

### 1. JSON va XMLHttpRequest (`first/`)

JSON formati va HTTP so'rovlarini yuborish usullari o'rgatiladi.

**Mavzular:**
- `JSON.parse()` va `JSON.stringify()` orqali ob'ektni klonlash
- `XMLHttpRequest` orqali API so'rovlari
- Tashqi JSON faylidan valyuta kurslari bilan ishlash

---

### 2. LocalStorage (`localStorage/`)

Brauzer `localStorage` API dan foydalanib ma'lumotlarni lokal saqlash o'rgatiladi.

**Mavzular:**
- `localStorage.setItem()` — ma'lumot yozish
- `localStorage.getItem()` — ma'lumot o'qish
- Forma ma'lumotlarini qayta tiklash (persistence)
- Sahifa mavzusini (dark/light theme) saqlash

---

### 3. Fetch API (`third-json_placeholder/`)

Zamonaviy `fetch()` API va REST so'rovlari bilan ishlash o'rgatiladi.

**Mavzular:**
- `fetch()` yordamida GET va POST so'rovlari
- [JSONPlaceholder](https://jsonplaceholder.typicode.com/) API bilan ishlash
- `.then()` / `.catch()` zanjiri
- Dinamik HTML elementlar yaratish

---

### 4. Promislar (`Promises/`)

Asinxron dasturlashning asosi bo'lgan Promise tushunchasi o'rgatiladi.

**Mavzular:**
- `new Promise(resolve, reject)` orqali promis yaratish
- `.then()`, `.catch()`, `.finally()` metodlari
- `Promise.all()` orqali bir nechta asinxron operatsiyani boshqarish

---

### 5. Array Metodlari (`Array-methods/`)

Massivlar ustida amal bajaruvchi funksional yuqori darajali metodlar o'rgatiladi.

**Mavzular:**
- `.map()` — har bir elementni yangi qiymatga o'zgartirish
- `.filter()` — shartga mos elementlarni ajratib olish
- `.every()` — barcha elementlar shartni qanoatlantiradimi?
- `.some()` — kamida bitta element shartni qanoatlantiradimi?
- `.reduce()` — elementlarni birlashtirish yoki yig'indi hisoblash

---

### 6. Regulyar Ifodalar (`Regular-expression/`)

Satrlarni tekshirish va qidirish uchun regex (regular expression) pattern'lari o'rgatiladi.

**Mavzular:**
- Regex sintaksisi va pattern'lar
- Satr mosligi va qidirish
- Webpack yordamida modul tizimi sozlash
